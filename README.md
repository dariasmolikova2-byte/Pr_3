Аналітичне опрацювання контекстної діаграми A-0 'Розклад занять' 

Функціональний блок «Розклад занять» описує процес, який перетворює сукупність вхідних даних (Дані про Групи, Аудиторії, Викладачів тощо) у Сформований розклад. 

Критичне значення має Керуючий вплив (Control), який визначає, як саме відбувається трансформація. Керуючі стрілки (Положення про розклад, Навчальний план) містять усі правила та обмеження (наприклад, заборону конфліктів, принцип цілих потоків), необхідні для складання коректного розкладу. 

Механізм (Mechanism) – це ресурси, що забезпечують виконання роботи: як людські (Персонал), так і технологічні (Програмне забезпечення). 

Аналіз показує, що критичною залежністю є C2: Положення про розклад. Будь-яка нечіткість у цьому документі призведе до некоректного розкладу, навіть якщо всі вхідні дані (I) є точними.
